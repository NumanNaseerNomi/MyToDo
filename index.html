    Firebase is Google's free cloud database — sign in with your Google account and paste one URL below. Your tasks will sync across every device instantly. Free tier = 1 GB storage, 10 GB/month transfer.

    1
    Open Firebase Console
    Go to console.firebase.google.com and sign in with your Google account.
    2
    Create a project
    Click Add project → give it a name (e.g. my-tasks) → Continue → Create project.
    3
    Enable Realtime Database
    In the left menu: Build → Realtime Database → Create database. Pick any region. Choose Start in test mode → Enable.
    4
    Copy & paste the URL below
    At the top of the Realtime Database page you'll see a URL like:
    https://my-tasks-xxxx-default-rtdb.firebaseio.com
    Copy it and paste below. Done!
